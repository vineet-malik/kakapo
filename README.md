# Kakapo

Voice-native collaborative IDE built on Zed + ACP.

One session, one session-scoped coding agent, one master at a time. Audio stays on the master's device; tool calls run against the master's local workspace. BYOK for LLM inference — we don't resell tokens, we make yours go further.
